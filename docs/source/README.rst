xmldirector.dropbox
===================

Integration of 

- Plone (https://www.plone.org)
- XML Director (https://www.xml-director.info) 
- Dropbox

Requirements
------------

- Plone 4.3 (tested)
  
- Plone 5.0 (experimental, in progress)

- XML Director (xmldirector.plonecore)
